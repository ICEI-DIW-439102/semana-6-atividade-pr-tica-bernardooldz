## Proposta do Projeto

Desenvolvimento de uma aplicação web chamada **iBook**, que funciona como uma plataforma digital voltada para escritores independentes.

A proposta é criar um ambiente onde autores possam divulgar sobre sua história e suas obras e leitores possam descobrir novos conteúdos e autores, promovendo interação e crescimento dentro da comunidade.

---

## Breve Descrição do Projeto

O **iBook** é uma plataforma inspirada em redes sociais e aplicativos de leitura, com foco em:

* Divulgação de obras por autores independentes
* Descoberta de novos conteúdos por leitores
* Interface moderna e acessível
* Experiência de leitura agradável

O projeto utiliza **HTML e CSS**, com foco em boas práticas de estrutura semântica, organização de layout e criação de um design system consistente.
